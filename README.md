# Java CRM System

This is a simple Customer Relationship Management (CRM) project built using Java.

The main purpose of this project is to practice Object-Oriented Programming (OOP) concepts and Java Collections Framework. It allows users to manage customer records through a menu-driven console application.

## Features

* Add Customer
* View All Customers
* Search Customer by Name
* Delete Customer by ID
* View Customer Statistics

## Technologies Used

* Java
* OOP Concepts
* ArrayList
* HashMap
* Git & GitHub

## Project Files

```text
Customer.java
CRMSystem.java
Main.java
```

## How It Works

Each customer is stored as an object of the Customer class.

* ArrayList is used to store all customer records.
* HashMap is used for faster customer lookup using ID.
* The application runs through a simple menu-driven interface.

## Concepts Practiced

* Classes and Objects
* Constructors
* Encapsulation
* ArrayList
* HashMap
* Loops and Conditional Statements

## How to Run

Compile the files:

```bash
javac Customer.java CRMSystem.java Main.java
```

Run the program:

```bash
java Main
```
